# CCL Practical Exam Task 2026

Use this repo to perform the experiments. The Experiment Number will be present on the top of your answer sheet. There are three different apps in three folders strating with recipe.  

## How to Use this Repo
- Open this repo on the browser.
- Download the code as zip file.
- Based on the experiment assigned to you you have to create your own repo in your github account and upload the contents of the corresponding folder to your repo.
- Eg.: If you Exp is 1 then you have to create a new repo and upload the contents inside `recipe-local` to your repo.
- Then deploy using the repo that you have created. 
- After deployment show it to the supervisor and write the deployment steps on the answer sheets.

## Experiments Aim
1. **VB** : Deploy the `recipe-static` website on VirtualBox. The website must run on PORT 8080 on the host. 
2. **VB** : Deploy the `recipe-local` app on VirtualBox. The app must run on PORT 8081 on the host. 
3. **EC2** : Deploy the `recipe-static` website on EC2. The website must run on PORT 80. 
4. **EC2** : Deploy the `recipe-local` app on EC2. The app must run on PORT 3333. 
5. **EB** : Deploy the `recipe-local` app on Elastic Beanstalk. The app must run on PORT 3000. 
6. **RDS** : Deploy the `recipe-db` app on EC2 & RDS. The app must run on PORT 5555 in EC2 and connected to RDS. Configuration must be done using Putty.
7. **S3** : Deploy the `recipe-static` app on S3.
8. **Docker** : Deploy the `recipe-local` app on Docker using Dockerfile and docker run commands. The app must run on PORT 8082 on the host machine. Base image to be used is `node:alpine`.
9.  **Docker** : Deploy the `recipe-static` app on Docker using docker run commands. The app must run on PORT 8082 on the host machine.
